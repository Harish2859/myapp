# MyApp - React + Docker + Jenkins

This is a DevOps project demonstrating CI/CD with GitHub webhooks, Jenkins, and Docker.

## Features
- React application with Vite
- Docker containerization
- Jenkins CI/CD pipeline
- GitHub webhook integration

## Build Status
Testing Jenkins webhook integration

## Getting Started

### Prerequisites
- Node.js 16+
- Docker
- Jenkins

### Installation
```bash
npm install
npm run build
```

### Docker Build
```bash
docker build -t react-app:latest .
```
